## 📁 Command Category: Suggestion

### 💾 `/suggestion`

**Description:** Suggestion commands

### 💻 Usage

`/suggestion setup <channel>`
`/suggestion accept <message_id>`
`/suggestion decline <message_id>`
`/suggestion send <saran>`
`/suggestion list`
